> superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax for tracking. Read

**Goal:** Migrate the SSH tab's add/edit/detail flow from centered popup modals onto the
new side-pane form substrate — `ListCursor`-driven host list with a synthetic "+ add new"
row, `FormPane`-based add/edit form, a read-only Info + Editable inspector side pane,
background-open via `TabManager::push_background`, and context-aware footer hints.
Connection/SFTP/PTY behavior is **not** changed. Secrets/key handling stays as today.

**Architecture:** All form spec construction moves from `ssh_new_modal` / `ssh_edit_modal`
(which build `ModalSpec`) into new `ssh_add_form_spec` / `ssh_edit_form_spec` functions
(which build `FormSpec`). `SshWidget` gains a `ListCursor` field replacing the bare
`selected_idx` in `SshState`. A new `SshInspector` struct (pure data, no ratatui) holds
the Info-section facts about the selected host; it is rendered by the wire layer via
`render_form_pane` when no form is open. The host-list render path reads `ListCursor`
instead of `SshState::selected_idx` directly. `ssh_modal_for_key` is retired for `N`/`E`;
other modal-based flows (`G`, `S`, `K`, `X`, `F`) remain unchanged — they are out of scope.
`wire.rs` changes are fully additive.

**Tech Stack:** Rust, ratatui (sid-widgets only), `sid_widgets::form::{FormSpec, FormPane,
FormValues, SectionKind, Validate}`, `sid_widgets::list_cursor::{ListCursor, CursorTarget}`,
`sid_widgets::split_view::SplitView`, `sid_core::tab::TabManager::push_background`,
`sid_core::event::KeyChord::{strip_nav, is_background_open}`, insta snapshots.

fn render_host_list(&self, frame: &mut Frame<'_>, rect: Rect, theme: &Theme) {
    let cursor = self.state.cursor();
    let hosts = self.state.visible_hosts();
    let mut lines: Vec<Line<'_>> = Vec::new();

    // Synthetic "+ add new" row (only when cursor.add_new).
    if cursor.add_new {
        let is_sel = cursor.target() == CursorTarget::AddNew;
        let marker = if is_sel { '>' } else { ' ' };
        let style = if is_sel {
            Style::default()
                .fg(theme.accent_primary.into())
                .add_modifier(Modifier::BOLD)
        } else {
            Style::default().fg(theme.accent_primary.into())
        };
        lines.push(Line::from(Span::styled(
            format!("{marker} + add new"),
            style,
        )));
    }

    if hosts.is_empty() && !cursor.add_new {
        lines.push(Line::from(Span::styled(
            "  (no hosts configured)",
            Style::default().fg(theme.muted.into()),
        )));
    } else {
        for (i, h) in hosts.iter().enumerate() {
            let is_selected = cursor.target() == CursorTarget::Item(i);
            let dot = if is_selected { '●' } else { '○' };
            let marker = if is_selected { '>' } else { ' ' };
            let suffix = if h.source == SshHostSource::SshConfig { " (cfg)" } else { "" };
            let label = format!("{marker} {dot} {}{suffix}", h.alias);
            let style = if is_selected {
                Style::default()
                    .fg(theme.accent_primary.into())
                    .add_modifier(Modifier::BOLD)
            } else {
                Style::default().fg(theme.foreground.into())
            };
            lines.push(Line::from(Span::styled(label, style)));
        }
    }

    let focused = self.focused_pane == SshFocus::Hosts;
    let border_color = if focused { theme.accent_primary } else { theme.muted };
    let mut title_style = Style::default().fg(theme.foreground.into());
    if focused { title_style = title_style.add_modifier(Modifier::BOLD); }
    let block = Block::default()
        .borders(Borders::ALL)
        .border_style(Style::default().fg(border_color.into()))
        .title(" Hosts ")
        .title_style(title_style);
    frame.render_widget(Paragraph::new(lines).block(block), rect);
}

### Task 2: `SshInspector` — Info section data + `FormSpec` builder

**Files:**

- Modify: `crates/sid-widgets/src/ssh.rs` (add `SshInspector` struct + `impl` + builder fn)

The inspector is the read-only right-pane view when a host is selected and no form is open.
It surfaces two logical sections:
- **Info** (`SectionKind::Info`): resolved connection facts — alias, host, port, user,
  auth method, source (Manual vs SSH Config), last connected (epoch → human string),
  last SFTP path.
- **Editable** (`SectionKind::Editable`): identity_file only (the only per-host pref that
  makes sense to surface here; auth_kind is editable in the add/edit form — the inspector
  shows it read-only in Info, per decision 2).

The inspector uses `FormSpec` / `FormField` / `SectionKind` from the substrate.

- [x] **Step 1: Define `SshInspector` and the spec builder**

```rust
use crate::form::{FormField, FormId, FormSection, FormSpec, FormValues, SectionKind};
use crate::modal::Field; // FormField wraps modal::Field for payload

/// Read-only + prefs view for a selected SSH host.
///
/// # Examples
///
/// ```
/// use sid_store::{SshAuthKind, SshHost, SshHostSource};
/// use sid_widgets::ssh::SshInspector;
/// let h = SshHost {
///     alias: "dev".into(), host: "10.0.0.1".into(), port: 22,
///     user: "alice".into(), identity_file: None,
///     source: SshHostSource::Manual, last_connected: 0,
///     command_history: vec![], last_sftp_path: None,
///     auth_kind: SshAuthKind::Agent,
/// };
/// let insp = SshInspector::from_host(&h);
/// assert_eq!(insp.alias, "dev");
/// ```
#[derive(Clone, Debug)]
pub struct SshInspector {
    pub alias: String,
    pub host: String,
    pub port: u16,
    pub user: String,
    pub identity_file: Option<String>,
    pub source: SshHostSource,
    pub last_connected: u64,
    pub last_sftp_path: Option<String>,
    pub auth_kind: SshAuthKind,
}

impl SshInspector {
    /// Build an inspector from a host record. Copies all fields.
    pub fn from_host(h: &SshHost) -> Self {
        Self {
            alias: h.alias.clone(),
            host: h.host.clone(),
            port: h.port,
            user: h.user.clone(),
            identity_file: h.identity_file.clone(),
            source: h.source,
            last_connected: h.last_connected,
            last_sftp_path: h.last_sftp_path.clone(),
            auth_kind: h.auth_kind,
        }
    }

    /// Format `last_connected` epoch seconds as a human-readable string.
    /// 0 → "never".
    pub fn last_connected_display(&self) -> String {
        if self.last_connected == 0 {
            "never".to_string()
        } else {
            format!("{} s ago", self.last_connected)
        }
    }

    /// Build a `FormSpec` suitable for passing to `render_form_pane` as the
    /// inspector side pane. The `Editable` section contains only `identity_file`
    /// so the user can set a key path; all other facts live in `Info`.
    ///
    /// Form id: `"ssh.inspect:<alias>"` — the submit handler in wire.rs
    /// updates only `identity_file` when submitted.
    pub fn to_form_spec(&self) -> FormSpec {
        use crate::form::{FormField, FormSection, SectionKind, Validate};
        use crate::modal::Field;
        let info_section = FormSection {
            title: "Host".to_string(),
            kind: SectionKind::Info,
            fields: vec![
                FormField {
                    key: "alias".to_string(),
                    field: Field::Display {
                        label: "Alias".to_string(),
                        body: self.alias.clone(),
                    },
                    validators: vec![],
                },
                FormField {
                    key: "host".to_string(),
                    field: Field::Display {
                        label: "Host".to_string(),
                        body: format!("{}:{}", self.host, self.port),
                    },
                    validators: vec![],
                },
                FormField {
                    key: "user".to_string(),
                    field: Field::Display {
                        label: "User".to_string(),
                        body: self.user.clone(),
                    },
                    validators: vec![],
                },
                FormField {
                    key: "auth".to_string(),
                    field: Field::Display {
                        label: "Auth".to_string(),
                        body: match self.auth_kind {
                            SshAuthKind::Agent => "agent".to_string(),
                            SshAuthKind::Key => "key".to_string(),
                            SshAuthKind::Password => "password".to_string(),
                        },
                    },
                    validators: vec![],
                },
                FormField {
                    key: "source".to_string(),
                    field: Field::Display {
                        label: "Source".to_string(),
                        body: match self.source {
                            SshHostSource::Manual => "manual".to_string(),
                            SshHostSource::SshConfig => "~/.ssh/config".to_string(),
                        },
                    },
                    validators: vec![],
                },
                FormField {
                    key: "last_connected".to_string(),
                    field: Field::Display {
                        label: "Last connected".to_string(),
                        body: self.last_connected_display(),
                    },
                    validators: vec![],
                },
                FormField {
                    key: "last_sftp_path".to_string(),
                    field: Field::Display {
                        label: "Last SFTP path".to_string(),
                        body: self.last_sftp_path
                            .clone()
                            .unwrap_or_else(|| "(none)".to_string()),
                    },
                    validators: vec![],
                },
            ],
        };
        // Only Manual hosts have an editable identity_file; SSH-Config hosts
        // are read-only (the file already stores it).
        let editable_section = if self.source == SshHostSource::Manual {
            Some(FormSection {
                title: "Preferences".to_string(),
                kind: SectionKind::Editable,
                fields: vec![FormField {
                    key: "identity_file".to_string(),
                    field: Field::Text {
                        label: "Identity file".to_string(),
                        value: self.identity_file.clone().unwrap_or_default(),
                        placeholder: Some("~/.ssh/id_ed25519".to_string()),
                        secret: false,
                    },
                    validators: vec![],
                }],
            })
        } else {
            None
        };
        let mut sections = vec![info_section];
        if let Some(prefs) = editable_section {
            sections.push(prefs);
        }
        FormSpec {
            id: FormId(format!("ssh.inspect:{}", self.alias)),
            title: format!("SSH · {}", self.alias),
            sections,
            reshape: None,
        }
    }
}

```rust
/// Handle SSH-tab keys that open a side-pane `FormPane` rather than a modal.
///
/// Returns `true` when a form was opened (the caller should skip the
/// `ssh_modal_for_key` branch).
///
/// Covers:
/// - `N` — open the Add Host form.
/// - `E` — open the Edit Host form for the selected Manual host.
/// - `→` / `Enter` on a host row (not add-new) — open the inspector pane.
///
/// Does not handle `G`, `S`, `K`, `X`, `F` — those remain modal.
pub fn dispatch_ssh_form_key(
    sid_app: &mut SidApp,
    chord: sid_core::event::KeyChord,
) -> bool {
    use crossterm::event::KeyCode;
    use sid_store::SshHostSource;
    use sid_widgets::ssh::{ssh_add_form_spec, ssh_edit_form_spec, SshInspector};

    match chord.code {
        KeyCode::Char('N') | KeyCode::Char('n') => {
            open_form(sid_app, ssh_add_form_spec());
            true
        }
        KeyCode::Char('E') | KeyCode::Char('e') => {
            let Some(host) = ssh_selected_host(sid_app) else { return false; };
            // ssh-config entries are read-only.
            if host.source == SshHostSource::SshConfig {
                return false;
            }
            open_form(sid_app, ssh_edit_form_spec(&host));
            true
        }
        KeyCode::Right => {
            // → from list focus opens the inspector side pane for the selected host.
            let Some(host) = ssh_selected_host(sid_app) else { return false; };
            let spec = SshInspector::from_host(&host).to_form_spec();
            open_form(sid_app, spec);
            true
        }
        _ => false,
    }
}
