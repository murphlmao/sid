**Goal:** Replace the Network tab's read-only detail modal (currently `ModalSpec` + `Field::Display`
rows opened by `Enter` on a focused interface) with a real UX-v2 side pane: an Info section showing
the interface's observable data plus an Editable section for the two sid-level prefs — pin-to-top
and display alias. Pin-to-top reorders the interfaces list (pinned always sorts before unpinned);
alias replaces the interface name in the sidebar display. Both prefs persist via `Store::put_setting`
using a per-interface key scheme. A one-line muted OS-config disclaimer appears at the bottom of the
pane. Focus follows master decision 4: `→` / `Enter` enters the pane, `Esc` / `←` leaves (with
discard-confirm when dirty); footer adapts per focus state.

**Architecture:** `crates/sid-widgets/src/network/` gains a new file `detail_pane.rs` that builds
and updates the `FormSpec` for a given `NetInterface` plus a `NetInterfacePrefs` struct carrying the
two editable prefs. `NetworkWidget` grows a `SplitView<DetailView>` (from the substrate) that
replaces the existing `Focus::Interfaces` drill-in stub. The wire layer (`crates/sid/src/wire.rs`)
is updated in three additive places: `SidApp` gains no new fields (the `form` / `form_origin_tab`
fields from substrate Task 9 are reused); `dispatch_form_submit` gets a new match arm for
`"network.interface_prefs"` that reads the submitted `FormValues`, writes two settings keys, and
calls a new `apply_network_prefs` helper; `network_modal_for_key` is retired — `Enter` now calls
`open_form` instead.

**Tech Stack:** Rust, ratatui (sid-widgets only), crossterm, substrate `FormSpec` / `FormPane` /
`SplitView` / `render_form_pane`, `Store::put_setting` + `TypedSettings`, insta snapshots.

### Settings key scheme

Per-interface prefs live in the `settings` table under these key patterns:

| Pref | Key | Value encoding |
|---|---|---|
| Pin-to-top | `network.iface.<name>.pinned` | `"true"` / `"false"` (via `TypedSettings::put_bool`) |
| Display alias | `network.iface.<name>.alias` | UTF-8 string (via `TypedSettings::put_string`); empty string means "no alias" |

Two new constants are added to `sid_store::settings_keys`:
```
pub const NETWORK_IFACE_PINNED_PREFIX: &str = "network.iface.";
// full key = format!("{}{}. pinned", NETWORK_IFACE_PINNED_PREFIX, iface_name)
// full key = format!("{}{}. alias",  NETWORK_IFACE_PINNED_PREFIX, iface_name)
```

(In practice the code uses inline `format!` — the constants serve as doc anchors and test fixtures.)

### Alias display rule

The sidebar renders the alias **instead of** the raw interface name, with the raw name appended
in muted parentheses: `eth0 (work-lan)`. If no alias is set the raw name is shown as before. This
matches the "alongside" spirit of decision 9 while keeping the raw name always visible for
debugging, and avoids a separate "alias" column.

### Pin-to-top sort rule

`InterfacesSidebarState::set_data_with_default_route` currently sorts by `iface_sort_score`. Pinned
interfaces receive a score bonus of `−1000` (i.e. their score is clamped to `score.saturating_sub(1000)`)
applied **before** the existing default-route / up / virtual bonuses, so pinned always wins. The
widget loads prefs from the store at `apply_snapshot` time and passes a `&[&str]` pinned-names slice
into the sidebar's updated `set_data_with_prefs` method.

//! Detail-pane spec builder and per-interface prefs type for the Network tab.
//!
//! `build_form_spec` constructs a [`FormSpec`] from a live [`NetInterface`]
//! snapshot and a set of user prefs; the wire layer opens the form and
//! dispatches the submit back here via [`prefs_from_values`].

use sid_core::adapters::sys::NetInterface;
use sid_widgets::form::{
    FormField, FormId, FormSection, FormSpec, FormValues, SectionKind, Validate,
};

// Re-export the modal Field enum that FormSpec uses for payloads.
use sid_widgets::modal::Field;

/// The two sid-level per-interface preferences.
///
/// # Examples
///
/// ```
/// use sid_widgets::network::detail_pane::NetInterfacePrefs;
/// let p = NetInterfacePrefs { pinned: true, alias: "work-lan".into() };
/// assert!(p.pinned);
/// assert_eq!(p.alias, "work-lan");
/// ```
#[derive(Debug, Clone, Default, PartialEq, Eq)]
pub struct NetInterfacePrefs {
    /// When `true`, this interface sorts before all unpinned interfaces.
    pub pinned: bool,
    /// Human-readable alias displayed in the sidebar. Empty = no alias.
    pub alias: String,
}

/// Build the `FormSpec` for the network interface detail pane.
///
/// * The `Info` section shows observable data from the current probe snapshot.
/// * The `Editable` section holds the two user prefs.
/// * The trailing disclaimer is an additional `Info` section with one
///   `Display` field.
///
/// # Examples
///
/// ```
/// use sid_core::adapters::sys::NetInterface;
/// use sid_widgets::network::detail_pane::{build_form_spec, NetInterfacePrefs};
///
/// let iface = NetInterface {
///     name: "eth0".into(),
///     addrs: vec!["192.168.1.2".into()],
///     rx_bytes: 1024,
///     tx_bytes: 512,
///     is_up: true,
/// };
/// let prefs = NetInterfacePrefs { pinned: false, alias: String::new() };
/// let spec = build_form_spec(&iface, &prefs, false);
/// assert_eq!(spec.id.0, "network.interface_prefs");
/// // Two sections: Info + Editable + disclaimer = 3.
/// assert_eq!(spec.sections.len(), 3);
/// ```
pub fn build_form_spec(
    iface: &NetInterface,
    prefs: &NetInterfacePrefs,
    is_default_route: bool,
) -> FormSpec {
    let status_label = if iface.is_up { "up" } else { "down" };
    let default_badge = if is_default_route { " (default route)" } else { "" };
    let status_str = format!("{status_label}{default_badge}");

    let addrs_str = if iface.addrs.is_empty() {
        "(none)".to_string()
    } else {
        iface.addrs.join(", ")
    };

    let rx_str = format_bytes(iface.rx_bytes);
    let tx_str = format_bytes(iface.tx_bytes);

    let info_section = FormSection {
        kind: SectionKind::Info,
        title: Some("Interface".into()),
        fields: vec![
            FormField {
                key: "status".into(),
                label: "Status".into(),
                field: Field::Display {
                    label: "Status".into(),
                    body: status_str,
                },
                validate: None,
            },
            FormField {
                key: "addresses".into(),
                label: "Addresses".into(),
                field: Field::Display {
                    label: "Addresses".into(),
                    body: addrs_str,
                },
                validate: None,
            },
            FormField {
                key: "rx".into(),
                label: "RX".into(),
                field: Field::Display {
                    label: "RX".into(),
                    body: rx_str,
                },
                validate: None,
            },
            FormField {
                key: "tx".into(),
                label: "TX".into(),
                field: Field::Display {
                    label: "TX".into(),
                    body: tx_str,
                },
                validate: None,
            },
        ],
    };

    let edit_section = FormSection {
        kind: SectionKind::Editable,
        title: Some("sid prefs".into()),
        fields: vec![
            FormField {
                key: "pinned".into(),
                label: "Pin to top".into(),
                field: Field::Toggle { label: "Pin to top".into(), value: prefs.pinned },
                validate: None,
            },
            FormField {
                key: "alias".into(),
                label: "Display alias".into(),
                field: Field::Text {
                    label: "Display alias".into(),
                    value: prefs.alias.clone(),
                    placeholder: Some("e.g. work-lan".into()),
                },
                validate: Some(Validate::MaxLen(40)),
            },
        ],
    };

    let disclaimer_section = FormSection {
        kind: SectionKind::Info,
        title: None,
        fields: vec![FormField {
            key: "os_note".into(),
            label: "".into(),
            field: Field::Display {
                label: "".into(),
                body: "OS-level interface configuration is not supported here.".into(),
            },
            validate: None,
        }],
    };

    FormSpec {
        id: FormId("network.interface_prefs".into()),
        title: format!("Interface: {}", iface.name),
        sections: vec![info_section, edit_section, disclaimer_section],
        reshape: None,
        watched_keys: vec![],
    }
}

/// Extract [`NetInterfacePrefs`] from a submitted `FormValues` map.
/// Returns `None` if required keys are absent (should not happen in
/// practice since the form always initialises them).
///
/// `FormValues` is `BTreeMap<String, String>`. The `pinned` key holds
/// `"true"` or `"false"` (as produced by the substrate's `Toggle` field
/// via `value_string()`); anything else returns `None`. The `alias` key
/// is trimmed.
///
/// # Examples
///
/// ```
/// use std::collections::BTreeMap;
/// use sid_widgets::network::detail_pane::prefs_from_values;
///
/// let mut map: BTreeMap<String, String> = BTreeMap::new();
/// map.insert("pinned".to_string(), "true".to_string());
/// map.insert("alias".to_string(), "home".to_string());
/// let prefs = prefs_from_values(&map).unwrap();
/// assert!(prefs.pinned);
/// assert_eq!(prefs.alias, "home");
/// ```
pub fn prefs_from_values(values: &FormValues) -> Option<NetInterfacePrefs> {
    let pinned = match values.get("pinned")?.as_str() {
        "true" => true,
        "false" => false,
        _ => return None,
    };
    let alias = values.get("alias")?.trim().to_string();
    Some(NetInterfacePrefs { pinned, alias })
}

/// Helper: format bytes as a short human string (KB / MB / GB).
fn format_bytes(b: u64) -> String {
    const KB: u64 = 1_024;
    const MB: u64 = KB * 1_024;
    const GB: u64 = MB * 1_024;
    if b >= GB {
        format!("{:.1} GB", b as f64 / GB as f64)
    } else if b >= MB {
        format!("{:.1} MB", b as f64 / MB as f64)
    } else if b >= KB {
        format!("{:.1} KB", b as f64 / KB as f64)
    } else {
        format!("{b} B")
    }
}

/// assert_eq!(pinned_key("wlan0"), "network.iface.wlan0.pinned");

    /// Like [`Self::set_data_with_default_route`] but also accepts a slice of
    /// interface names that should sort before all unpinned interfaces.
    ///
    /// Selection is preserved across refreshes by interface name.
    ///
    /// # Examples
    ///
    /// ```
    /// use sid_core::adapters::sys::NetInterface;
    /// use sid_widgets::network::interfaces_sidebar::InterfacesSidebarState;
    ///
    /// let mut s = InterfacesSidebarState::new();
    /// let data = vec![
    ///     NetInterface { name: "eth0".into(), addrs: vec![], rx_bytes: 0, tx_bytes: 0, is_up: true },
    ///     NetInterface { name: "wlan0".into(), addrs: vec![], rx_bytes: 0, tx_bytes: 0, is_up: true },
    /// ];
    /// s.set_data_with_prefs(data, Some("wlan0"), &["eth0"]);
    /// // Pinned eth0 must be first despite wlan0 holding the default route.
    /// assert_eq!(s.rows()[0].name, "eth0");
    /// ```
    pub fn set_data_with_prefs(
        &mut self,
        mut data: Vec<NetInterface>,
        default_route: Option<&str>,
        pinned: &[&str],
    ) {
        sort_interfaces_with_pins(&mut data, default_route, pinned);
        self.default_route = default_route.map(|s| s.to_string());
        let prev_name = self.data.get(self.selected).map(|i| i.name.clone());
        self.data = data;
        self.selected = prev_name
            .and_then(|n| self.data.iter().position(|i| i.name == n))
            .unwrap_or(0);
        if self.selected >= self.data.len() {
            self.selected = 0;
        }
    }

pub fn sort_interfaces_with_pins(
    data: &mut [NetInterface],
    default_route: Option<&str>,
    pinned: &[&str],
) {
    data.sort_by(|a, b| {
        let a_pinned = pinned.contains(&a.name.as_str());
        let b_pinned = pinned.contains(&b.name.as_str());
        match (a_pinned, b_pinned) {
            (true, false) => std::cmp::Ordering::Less,
            (false, true) => std::cmp::Ordering::Greater,
            (true, true) => a.name.cmp(&b.name),
            (false, false) => {

                let sb = iface_sort_score(b, default_route);
                sa.cmp(&sb).then_with(|| a.name.cmp(&b.name))
            }
        }
    });

/// Write submitted network interface prefs to the store and push the updated
/// alias + pinned state into the active network widget.
fn apply_network_prefs(
    sid_app: &mut SidApp,
    iface_name: &str,
    prefs: sid_widgets::network::detail_pane::NetInterfacePrefs,
) {
    use sid_store::TypedSettings;
    // Persist to store.
    let _ = sid_app
        .store
        .put_bool(
            &sid_widgets::network::detail_pane::pinned_key(iface_name),
            prefs.pinned,
        );
    let _ = sid_app
        .store
        .put_string(
            &sid_widgets::network::detail_pane::alias_key(iface_name),
            &prefs.alias,
        );

    // Push updated alias and pinned state into the live widget so the
    // sidebar re-renders immediately without waiting for the next probe tick.
    let tab = sid_app.app.tabs_mut().active_mut();
    if let Some(net) = tab
        .layout
        .iter_widgets_mut()
        .next()
        .and_then(|w| w.as_any_mut().downcast_mut::<sid_widgets::NetworkWidget>())
    {
        // Reload all aliases from the store (not just this one) so the
        // sidebar stays consistent across multiple edits in one session.
        let mut aliases = std::collections::HashMap::new();
        let mut pinned_names = std::collections::HashSet::new();
        use sid_store::Store;
        if let Ok(keys) = sid_app.store.list_setting_keys() {
            for key in &keys {
                if let Some(rest) = key.strip_prefix("network.iface.") {
                    if let Some(name) = rest.strip_suffix(".alias") {
                        if let Ok(Some(v)) = sid_app.store.get_string(key) {
                            aliases.insert(name.to_string(), v);

    /// interfaces, display aliases). Call this from wire.rs whenever a new
    /// probe tick arrives so the sidebar reflects current prefs.
    ///
    /// # Examples
    ///
    /// ```
    /// use sid_widgets::NetworkWidget;
    /// use sid_core::sys_probe::SysSnapshot;
    /// let mut w = NetworkWidget::new();
    /// w.apply_snapshot_with_prefs(
    ///     SysSnapshot::empty(),
    ///     std::collections::HashMap::new(),
    ///     std::collections::HashSet::new(),
    /// );
    /// ```
    pub fn apply_snapshot_with_prefs(
        &mut self,
        snap: sid_core::sys_probe::SysSnapshot,
        aliases: std::collections::HashMap<String, String>,
        pinned: std::collections::HashSet<String>,
    ) {
        self.ports.set_data(snap.listening_ports);
        self.procs.set_data(snap.processes);
        let pinned_refs: Vec<&str> = pinned.iter().map(|s| s.as_str()).collect();
        self.ifs.set_data_with_prefs(
            snap.interfaces,
            snap.default_route_iface.as_deref(),
            &pinned_refs,
        );
        self.ifs.set_aliases(aliases);
        self.ifs.set_pinned_names(pinned);
    }

pub fn refresh_network_widget(sid_app: &mut SidApp, snap: SysSnapshot) {
    let tab = sid_app.app.tabs_mut().active_mut();
    if tab.id.as_str() == "network" {
        if let Some(net) = tab
            .layout
            .iter_widgets_mut()
            .next()
            .and_then(|w| w.as_any_mut().downcast_mut::<sid_widgets::NetworkWidget>())
        {
            let (aliases, pinned) = load_all_network_prefs(sid_app.store.as_ref());
            net.apply_snapshot_with_prefs(snap, aliases, pinned);
        }
    }
}

/// Load all per-interface prefs from the store in one pass.
/// Returns (aliases map, pinned names set).
fn load_all_network_prefs(
    store: &impl sid_store::Store,
) -> (
    std::collections::HashMap<String, String>,
    std::collections::HashSet<String>,
) {
    use sid_store::TypedSettings;
    let mut aliases = std::collections::HashMap::new();
    let mut pinned = std::collections::HashSet::new();
    if let Ok(keys) = store.list_setting_keys() {
        for key in &keys {
            if let Some(rest) = key.strip_prefix("network.iface.") {
                if let Some(name) = rest.strip_suffix(".alias") {
                    if let Ok(Some(v)) = store.get_string(key) {
                        if !v.is_empty() {
                            aliases.insert(name.to_string(), v);
                        }
                    }
                }
                if let Some(name) = rest.strip_suffix(".pinned") {
                    if let Ok(Some(true)) = store.get_bool(key) {
                        pinned.insert(name.to_string());
                    }
                }
            }
        }
    }
    (aliases, pinned)
}
