Shown: a 3-step modal wizard (Algorithm → Path/Passphrase → Remote setup) for generating an SSH keypair from inside sid.
Step 1 picks Ed25519 by default and flags RSA-2048 as legacy; step 2 collects the output path, optional passphrase, and a comment;
step 3 displays the generated public key, lets you copy it, and offers to ssh-copy-id it into a known host using existing auth.
End state: keypair on disk at ~/.ssh/id_ed25519_prod, public key in the remote authorized_keys, host record updated — passwordless ssh.
